# ⚡ QUANTUM HEDGE AI - Trading Terminal

## AI аналитик рынка с TradingView графиками

### Что это:
- 📊 **TradingView графики** — крипта + форекс + акции + индексы
- 🤖 **AI чат** — анализирует рынок, даёт зоны покупки/продажи
- 🎯 **Визуальные зоны** — поддержка/сопротивление, тренды
- 📈 **Real-time данные** — прямо с TradingView

### Доступные инструменты:
- 🪙 Крипто: BTC, ETH, SOL, BNB, XRP, DOGE, ADA, AVAX
- 💱 Форекс: EUR/USD, GBP/USD, USD/JPY, AUD/USD
- 📈 Индексы: S&P 500, NASDAQ 100, Gold
- 🏢 Акции: Apple, Tesla, NVIDIA

### AI Команды:
- "Анализ" — полный технический анализ
- "Зоны" — зоны покупки/продажи
- "Тренд" — направление рынка
- "Прогноз" — куда пойдёт цена
- "Новости" — что влияет
- "Риски" — анализ рисков

## 🚀 Как запустить на Netlify:

1. Скачайте этот файл index.html
2. На Netlify: Sites → Add new site → Deploy manually
3. Перетащите папку TRADING-VIEW-TERMINAL
4. Готово!
