# 🚀 QUANTUM HEDGE - AI Trading Terminal

Профессиональный AI-powered Trading Terminal с графиками TradingView Lightweight Charts.

## ✨ Возможности

- 📊 **TradingView Lightweight Charts** - профессиональные графики
- 🤖 **AI Анализ** - автоматический поиск точек входа
- 🎤 **Голосовое управление** - голосовые команды
- 📈 **Real-time данные** - WebSocket подключение к Binance
- 🎯 **Авто-рисование** - линии поддержки/сопротивления, SL/TP
- 💹 **Trading Panel** - покупка/продажа в один клик
- 🌙 **Dark Theme** - стиль TradingView/Binance

## 🚀 Как использовать

### Способ 1: Локально (просто открыть в браузере)

```bash
# Скачать папку TRADING-VIEW-TERMINAL
# Открыть index.html в браузере (Chrome/Edge/Firefox)
```

⚠️ **Важно:** Из-за CORS политик некоторые браузеры блокируют API запросы при открытии файла напрямую. Используйте локальный сервер.

### Способ 2: Через локальный сервер (рекомендуется)

**Python:**
```bash
cd TRADING-VIEW-TERMINAL
python3 -m http.server 8080
# Открыть http://localhost:8080
```

**Node.js:**
```bash
cd TRADING-VIEW-TERMINAL
npx http-server -p 8080
# Открыть http://localhost:8080
```

**PHP:**
```bash
cd TRADING-VIEW-TERMINAL
php -S localhost:8080
```

### Способ 3: На своём сайте

1. Скопируйте папку `TRADING-VIEW-TERMINAL` на свой хостинг
2. Загрузите все файлы (index.html, styles.css, js/*)
3. Откройте через браузер по URL вашего сайта
4. Готово!

## 🎮 Команды

### Кнопки интерфейса:
- **🤖 AI Анализ** - полный анализ + рисование уровней
- **📈 Тренд** - линии EMA 20/50
- **📊 Поддержка** - уровни поддержки/сопротивления
- **🗑️ Очистить** - убрать все линии

### AI Чат / Голосовые команды:
- "Открой BTC" - сменить пару на BTCUSDT
- "Покажи ETH" - открыть ETHUSDT
- "Анализ" - сделать AI анализ
- "Сигнал" - получить торговый сигнал
- "Тренд" - показать текущий тренд
- "Уровни" - нарисовать поддержку/сопротивление
- "Очисти" - убрать все линии с графика

### Голосовое управление:
- Нажмите 🎤 в верхнем правом углу
- Скажите команду (например "Открой биткоин")
- AI выполнит и скажет результат

## 🛡️ Безопасность
